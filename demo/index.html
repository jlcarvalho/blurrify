Blurrify - CSS Motion Blur

jQuery Plugin

Download
Fork on GitHub

Blurrify is a jQuery Plugin based on Ross McMillan (@RossMcMillanNet) Motion Blur script.

Quick setup

After import jQuery and blurrify all we need to do is activate the Blurrify plugin.

$(document).ready(function(){
    $('.motion').blurrify();
});

Advanced Options

$(document).ready(function(){
    $('.motion').blurrify({
        blur: 10,           
        length: 15,         
        color: #fe0086      // Color  ("text", 
                            //         "background-color" or 
                            //         hexadecimal color)
    });
});
